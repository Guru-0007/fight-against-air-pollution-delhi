# Fight Against Air Pollution - Delhi 🌍

A modern, full-stack, dual-mode intelligence platform designed to actively track, analyze, and mitigate severe pollution levels directly across Delhi NCR.

Designed with an immersive glassmorphism aesthetic and genuine algorithmic calculation engines, this platform supports live community engagement (User Mode) alongside strict compliance and tracking tools (Government Mode).

## Features
- **Real-Time Data Streams**: Seamlessly aggregates dynamic European AQI scaling from Open-Meteo across exact geo-coordinates.
- **Government Command Center**: Built-in "Suspicion Engine" analyzing local wind dispersion, heat zones, and resident reports to hunt hidden factory polluters. Instantly drafts formal legal notices via an integrated mail handler directly citing the Air (Prevention and Control of Pollution) Act 1981.
- **Total Cost Calculator**: Analyzes hyper-local PM2.5 to estimate public health damages, asthma risks, lifespan impact, and yearly economic distress.
- **Blind Spot Mapping**: Automatically overlays missing coverage zones mapping >200 AQI regions against low citizen awareness volumes.
- **Secure Architecture**: Implements active ban/unban moderation and structured tracking.

## Running Locally

**Backend Server Setup**
```bash
cd backend
npm install
npm start
```

**Frontend Display**
The backend serves the application immediately at `http://localhost:3005`.
